**Rigid Body Modes**
There can be modes such as $[1,1]^T$ where there is no vibration, or there is but with infinite amplitude, such as two masses moving together in one direction. For these scenarios: $$\delta \textbf{q}=(C_1+C_2 t)\textbf{A}$$
For springs, when trying to find the matrix $\textbf{K}$, we can also ignore $2xL$ terms for some reason

If we had 2 masses only connected with a spring, we would have motion: $[\delta x_1, \delta x_2]^T=C_1 [1,-1]^T \cos(\sqrt{2k/m}t+\phi)+(C_2+C_3t)[1,1]^T$

Mode shapes are normally orthogonal

However in the case of the double pendulum they are not. Mode shapes are only orthogonal when the mass matrix $\textbf{M}$ is proportional the identity matrix and K is symmetric
It is always recommended to normalise your mode shapes, making them orthonormal

If for example we had an equation $[\delta x_1, \delta x_2]^T=C_1 \textbf{A}_1 \cos(\omega_1t+\phi)+C_2 \textbf{A}_2 \cos(\omega_2t+\phi)$
And we know that the initial condition is $[\delta x_1, \delta x_2]^T=[1,3]^T$

Then we can find the constants easily $C_1=[1,3]^T \cdot \textbf{A}_1$ and $C_2=[1,3]^T \cdot \textbf{A}_2$

**Small Amplitude Vibration around Unstable Equilibria Points**
If the stationary point is not a minimum, then some $\omega_i^2$ will be negative, giving $\omega_i$ as imaginary
Therefore the solutions change from being vibrations to being exponentially growing and decaying with time constant tau. $\textbf{A}e^{\pm t/\tau}$

**Large Amplitude Vibration**
We will have to look at integrations schemes, integrating the acceleration twice to get the position.
$\dot{\textbf{y}} = \textbf{f}(t,\textbf{y})$ where $\textbf{t}=[y_1,y_2]^T$

$\Delta t = t_{n+1}-t_n = \frac{t_b-t_a}{N}$

First order approximation: $\sum_{n=0}^{N-1}v(t_n)\Delta t$
Uses first order Taylor expansion around $t_n$
$v(t)=v(t_n)+(t-t_n)v'(t_n)+...$
$\int_{t_n}^{t_{n+1}}v(t)dt=\int_{t_n}^{t_{n+1}} v(t_n)+(t-t_n)v'(t_n)+... = v(t_n)\Delta t + \frac{1}{2}v'(t_n)\Delta t^2 +...$
We drop the $\Delta t^2$ term so error proportional to $N\Delta t^2$, so proportional to $\Delta t$, so first order

We can do this as a left Riemann Sum with $t_n$ or a right Riemann Sum with $t_{n+1}$
If the function is monotonically increasing, the left underestimates and the right overestimates

We could also do the mid-point Riemann Sum which is a second order method, or the Simpson's rule which uses a quadratic approximation to become a fourth order method.
If we want to reduce error by a factor of 100 with a fourth order method we only need to use $100^{1/4}$ times more intervals, or $\sqrt{10}$

If we have equations of motion in the form $\ddot{q}=f(\dot{q},q)$, in order to perform numerical integration we must rewrite as $\dot{y}=f(t,y)$ where $y = [q,\dot{q}]^T$

Forward Euler: First order integrator
$y(t_{n+1})=y(t_n)+f(y(t_n),t_n)\Delta t$
If we have a second order equation, we must numerically integrate $y$ and $\dot{y}$ at once

For more stability: backward Euler: $x(t_{n+1})=x(t_n)+v(t_{n+1})\Delta t$

Semi-implicit Euler: forward for velocity then backward for position

**Poincare Section**
Any two degree of freedom system's phase space is four dimensional ($\theta_1,\theta_2,\dot{\theta}_1, \dot{\theta}_2$)
To make a Poincare section, plot the state of the double pendulum every time $\theta_1=0$ for example
We get a fractal

Chaotic - small changes in initial conditions produce large changes in the final state
Time scale over which predictions can be made: Lyapunov time

**Time Varying Lagrangian**
If there is external driving, then $T+V$ is not constant
Include the time varying force as a time varying potential. Work backwards from the equation of motion to put it in the Lagrangian
$$M\ddot{q}+Kq=f\cos(\omega t)$$
The total solution is the complementary function plus the particular integral
$$q=\sum_i C_i \textbf{A}_i \cos(\omega_i t + \phi_i)+(K-\omega^2 M)^{-1}\textbf{f}\cos(\omega t)$$
If the driving force comes from internal change in the system we call it parametric resonance, and occurs when the internal parameter oscillates with a frequency $2\omega_0$
