A single neuron takes in a vector of inputs $\textbf{z}$ and computes the dot product with the weights vector $\textbf{w}$ to give the activity $a$. $a=\textbf{w}^T\textbf{z}=\sum_0^D w_d z_d$
The bias $w_0$ is included here as it's implicit that $z_0$ is 1

We then pass the activity into a non-linear activation function, a common one being the sigmoid function $x(a)=\frac{1}{1+exp(-a)}$

Changing the direction of the weight vector changes the direction of the sigmoid function. Increasing the magnitude of the weight vector increases the sharpness of the step.

The goal of training is to vary the weights so that this function outputs almost 1 for one class and almost 0 for another.
The classes are denoted by $y^{(n)}$

We can consider a loss function which looks like this: $G(\textbf{w})=-\sum_n [y^{(n)}log x(\textbf{z}^{(n)};\textbf{w})+(1-y^{(n)})log(1-x(\textbf{z}^{(n)};\textbf{w}))]$

In information theory the surprise of an outcome is $-log$ p(outcome) when observing $y^{(n)}$
This is also the relative entropy between $x(\textbf{z}^{(n)};\textbf{w})$ and $y^{(n)}$
