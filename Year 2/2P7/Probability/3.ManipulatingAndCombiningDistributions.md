**Functions of random variables**
Discrete random variables -

Let's say that we had $Y=g(X)$
$P_{Y=g(X)}(y)=\sum_{\{x:g(x)=y\}} P_X(x)$ where $\{x:g(x)=y\}$ is the preimage of y under g, so the values of x that go to the value of y.
If g is invertible (one-to-one) then $P_{Y=g(X)}(y)=P_X(g^{-1}(y))$

Continuous random variables -
Let $Y=g(X)$. Its CDF is $F_Y(y)=P[Y\leq y]=P[g(X)\leq y]$

If $g$ is strictly monotonic so one-to-one: $$f_{Y=g(X)}(y)=\frac{f_X(g^{-1}(y))}{|g'(g^{-1}(y))|}$$
Rule of thumb: A way to remember the change of variable formula is $|f_Y(y)dy|=|f_X(x)dx|$ so that $f_Y(y)=\frac{f_X(x)}{|\frac{dy}{dx}|}|_{x=g^{-1}(y)}$

For a non-monotonic transform, the PDF is the sum over all branches. It will most likely be symmetric or antisymmetric.

Affine transformation - 
Let $Y=aX + b$, then $f_Y(y)=\frac{1}{|a|}f_X(\frac{y-b}{a})$
$E[Y]=aE[X]+b$ and $Var[Y]=a^2 Var[X]$

**Sum of Random Variables**
Let $S=X+Y$
$E[S]=E[X]+E[Y]$
$Var[S]=Var[X]+Var[Y]+2Cov[X,Y]$ were $Cov[X,Y]=E[XY]-E[X]E[Y]$
The correlation coefficient - $\rho = \frac{Cov[X,Y]}{\sqrt{Var[X]Var[Y]}}$
$\rho > 0$ is positive correlation

Sum of discrete variables - 
By the law of total probability - $P_S(s)=\sum_{y\in Y}P[X+Y=s|Y=y]P_Y(y)$
This simplifies down to $P_S(s)=\sum_{y\in Y}P_X(s-y)P_Y(y)= \sum_{x\in X}P_Y(s-x)P_X(x)$
That is, $P_{X+Y}=P_X * P_Y$ (Discrete convolution)

Sum of continuous variables - 
X and Y continuous variables with joint PDF $f_{XY}(x,y)$

That is, $f_{X+Y}=f_X * f_Y$ (Convolution)

**Transforms of Distributions**
In probability we use generating functions to turn complicated operations into simpler ones, like Fourier transforms for signals.
Moments become derivates and the sum of independent variables come products

Probability-Generating Function (PGF): For discrete distributions:
$G_X(z)=\sum z^{k}P_X(k)=E[z^X]$

Moments -
Differentiating $E[z^X]$ and evaluating at $z=1$ gives:
$G_X^{(k)}(1)=E[X(X-1)...(X-k+1)]$
$E[X]=G'_X(1)$ and $Var[X]=G''_X(1)+G_X'(1)-G_X'(1)^2$
Sum of independent random variables -
$E[z^{X+Y}]=E[z^X]E[z^Y]\quad \Rightarrow \quad G_{X+Y}(z)=G_X(z)G_Y(z)$

Different PGFs of common distributions are in the data book

Moment-Generating Function (MGF): For continuous distributions:
$g_X(s)=\int_{-\infty}^{+\infty}f_X(x)e^{sx}dx=E[e^{sX}]$

Moments -
Differentiating $E[e^{sX}]$ and evaluating at $s=0$ gives:
$g_X^{(k)}(0)=E[X^k]$
$E[X]=g'_X(0)$ and $Var[X]=g''_X(0)-g_X'(0)^2$
Sum of independent random variables -
$E[e^{s(X+Y})]=E[e^{sX}]E[e^{sY}]\quad \Rightarrow \quad g_{X+Y}(z)=g_X(z)g_Y(z)$

Different MGFs of common distributions are in the data book

Consequences:
If $X$ ~ $B(n_x,p)$ and $Y$ ~ $B(n_Y,p)$ then $X+Y$ ~ $B(n_X+n_Y, p)$
If $X$ ~ $Pois(\lambda_X)$ and $Y$ ~ $Pois(\lambda_Y)$ then $X+Y$ ~ $Pois(\lambda_X+\lambda_Y)$
If $X$ ~ $N(\mu_X,\sigma_X^2)$ and $Y$ ~ $N(\mu_Y\sigma_Y^2)$ then $X+Y$ ~ $N(\mu_X+\mu_Y, \sigma_X^2+\sigma_Y^2)$
$g_{aX+b}(s)=E[e^{asX+bs}]=e^{bs}g_X(as)$

**The Central Limit Theorem**
If we have n independent random variables with finite mean and variance and we compute their centred and normalised sum. $Y_n=\frac{(X_1-\mu_1)+...+(X_n-\mu_n)}{\sqrt{\sigma_1^2+...+\sigma_n^2}}$

As $n\rightarrow \infty$, $f_{Y_n}\rightarrow N(0,1)$

We can approximate the Binomial distribution if n is sufficiently large: $X$ ~ $N(np,np(1-p))$

**Decision, Estimation and Hypothesis Testing**

If $X$ ~ $f_X(x;\theta)$, we wish to infer $\theta$ from observations
It is an estimation if $\theta$ takes values from a continuous set
It is a decision if if $\theta$ takes values from a discrete set 

Prior knowledge about $\theta$ is encoded by prior distribution $f_{\Theta}(\theta)$
Observation updates our belief about $\Theta$

We define:
Likelihood: $f_{X|\Theta}(x|\theta)$, Prior: $f_{\Theta}(\theta)$, Posterior: $f_{\Theta | X}(\theta | x)$
Posterior function obtained by Bayes theorem: $f_{\Theta | X}(\theta | x)=\frac{f_{X|\Theta}(x|\theta)f_{\Theta}(\theta)}{f_X(x)}$
Metrics to evaluate estimators:
* Unbiased $E[\theta(x)|\Theta = \theta]=\theta$
* Minimise variance
* Minimise mean squared error
Common estimators:
* Maximum likelihood estimator: chooses the parameter that makes the observed data most probable $\hat{\theta}(x)=argmax(f_{X|\Theta}(x|\theta))$
* Maximum a posterior MAP estimator: chooses the parameter that is most probable after observing the data
* Minimum mean squared error (MMSE)

**Hypothesis Testing**
1) Begin with the null hypothesis $H_0$
2) The p value is the probability of observing a result at least as extreme as the one obtained by assuming $H_0$ holds.
3) Choose a significance level $\alpha$
4) If $p\leq \alpha$ we reject $H_0$

If the question is: are boys taller than girls, the null hypothesis is that they are the same height.

If we were testing whether a coin is different from fair, we will have to consider both tails to find our p value
