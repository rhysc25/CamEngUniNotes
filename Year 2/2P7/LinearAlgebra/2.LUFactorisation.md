We can turn any matrix $\textbf{A}$ into $\textbf{A}=\textbf{L}\textbf{U}$
$\textbf{L}$ is a lower triangular matrix so it must be square, must have a leading diagonal of 1s, and anything above the ones are zeros.
$\textbf{U}$ is an upper echelon matrix, so everything below the leading diagonal is zeros. It would be called upper triangular if it was square. It has the shape of $\textbf{A}$

The process for the decomposition is very hard to explain, just do an example.

If we are solving $\textbf{A}\textbf{x}=\textbf{b}$, we can know equivalently solve $\textbf{L}(\textbf{U}\textbf{x})=\textbf{b}$
So first we solve for $\textbf{U}\textbf{x}$, then we solve for $\textbf{x}$
These solutions are very easy as the form of the matrices has removed the simultaneous aspect from the method, we can just use forward then backward substitution.

The first non-zero elements of $\textbf{U}$ in each row are called pivots.
The terms below the diagonal of $\textbf{L}$ are multipliers

If this fails as one of the elements is zero, we can do it still, just in a different order. This will give us $\textbf{A}=\hat{\textbf{L}}\textbf{U}$, where $\hat{\textbf{L}}$ is not lower triangular.
This is fine though, as we can just rearrange the rows with a permutation matrix $\textbf{L}=\textbf{P}\hat{\textbf{L}}$ 
This gives us: $$\textbf{A}=\hat{\textbf{L}}\textbf{U}=\textbf{P}^T\textbf{L}\textbf{U}\Rightarrow \textbf{P}\textbf{A}=\textbf{L}\textbf{U}$$
To reduce any change of difficulties, we can do partial pivoting, where we always scan the remaining non-zero elements in the next column in the remainder matrix to be zeroed and choosing the largest (in absolute value) to be used.

We want out pivots to be large, and our multipliers to be small. Partial pivoting stops this from occurring, so stops ill conditions from injecting into the problem.
However if the problem starts ill conditioned, e.g. a near singular matrix, then partial pivoting will not help

The number of independent rows = the number of independent columns for any matrix 
The number of independent columns is the rank of A

When we do LU decomposition, L will always be square and $L^{-1}$ will always exist
U will have an inverse if and only if A has an inverse

**The solution of $\textbf{A}\textbf{x}=\textbf{b}$**

If the equations are independent we have a unique solution
If the equations are compatible (i.e $\textbf{b}$ is in the column space of $\textbf{A}$) we get a solution of the form $\textbf{x}=\textbf{x}_0+\lambda \textbf{n}_1+\mu\textbf{n}_2+...$
If the equations are not compatible then there is no solutions 

We now must figure out two things, how to check whether $\textbf{b}$ is in the column space of $\textbf{A}$, and how to find the $\textbf{x}_0,\textbf{n}_1,...$

**How do we check whether b is in the column space?**

Before we said that we can do this $\textbf{L}(\textbf{U}\textbf{x})=\textbf{L}\textbf{c}=\textbf{b}$
We then see whether there is a solution for $\textbf{c}$, if there is then $\textbf{b}$ is in the column space of $\textbf{A}$ 

**How to find the $\textbf{x}_0,\textbf{n}_1,...$**

We then solve $\textbf{U}\textbf{x}=\textbf{c}$
If we have one too many unknowns for the number of equations, we can just write our answer in terms of the free variable.
The variables which have non zero pivots are called basic variables, but if it's zero they're called free variables.

As a general series of steps:
1) First, set the free variables to zero and solve to find $\textbf{x}_0$, the particular solution
2) Then, set the RHS to zero, and the free variables to 0 in turn, to find our vectors spanning the null space, $\textbf{x}_1, \textbf{x}_2,...$
3) This gives our general solution $\textbf{x}=\textbf{x}_0+t\textbf{x}_1+z\textbf{x}_2$
