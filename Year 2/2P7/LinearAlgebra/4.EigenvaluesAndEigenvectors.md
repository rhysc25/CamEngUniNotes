$A^k=U\Lambda^kU^{-1}$
$A^k\textbf{x}\approx \alpha_1 \lambda_1^k \textbf{u}_1$

The eigenvalues of a diagonal matrix are just the values down the leading diagonal
The eigenvalues of a triangular matrix are also just the values down the leading diagonal

The product of the eigenvalues is det(A) so a singular matrix must have one eigenvalue of 0

A defective matrix is one which does not have a full set of eigenvectors. This can only happen if it has a repeated eigenvalue.

Similar matrices have the same set of eigenvalues
$Ax=\lambda x \rightarrow M^{-1}A'M x = \lambda x \rightarrow A' y=\lambda y$
A matrix which is similar to a diagonal matrix is said to be diagonalisable
Any matrix with distinct eigenvalues can be diagonalised
A matrix with repeated roots may be diagonalisable, or it may not.

Inverse power method -
$Ax=\lambda x \rightarrow A^{-1}x=\frac{1}{\lambda}x$. Convergence factor $|\lambda_n|/|\lambda_{n-1}|$

Shifted inverse power method
$(A-\alpha I)x=Ax-\alpha I x \quad = \lambda x - \alpha x = (\lambda - \alpha)x$
We can obtain any eigenvalue. We will get the one closest to $\alpha$
Convergence factor $r=\frac{|\lambda_n - \alpha|}{|\lambda_{n-1}-\alpha|}$

**Singular Value Decomposition**

This is the next best thing, but for non-square matrices
By concentrating on the largest singular value, we focus on the most important part of A
The singular values come out in importance order

$A^TA$ is square and symmetric. $A^TA = Q\Lambda Q^T$
The singular values of A are $\sigma_i = \sqrt{\lambda_i}$ where $\lambda_i$ are the eigenvalues of $A^TA$ which also happens to be the eigenvalues of $AA^T$

The norm of A is equal to the largest singular value of A

We now define $\hat{q}_i$ or q circumflex as $\hat{q}\frac{Aq}{\sigma_i}$
The q circumflexes also form an orthonormal set
They are the eigenvectors of $AA^T$

$AA^T=\hat{Q}\hat{\Lambda}\hat{Q}^T$

Singular value decomposition: $A = \hat{Q} \Sigma Q^T$
Where $\Sigma$ is the matrix of singular values
