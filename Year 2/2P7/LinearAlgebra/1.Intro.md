The column space of $A$ is the set of $\textbf{f}$ for which $A\textbf{t}=\textbf{f}$
This is also the row space of $A^T$
The set of $\textbf{n}$ for which $A\textbf{n}=0$ is the null space
In the context of a set of bar forces, this would give a state of self stress
The left null space of A is the $\textbf{d}$ for which $A^T\textbf{d}=0$

A matrix is m x n. It has m rows and n columns
It transforms an n dimensional vector into an m dimensional vector, so maps $R^n$ to $R^m$

If we move to higher dimensions than 3, our dot product and magnitude still work, but the cross product doesn't, as we can't find a single direction $\hat{\textbf{n}}$ which is orthogonal to three vectors.

$A\textbf{x}=A(x\textbf{i}+y\textbf{j}+z\textbf{k})=x\textbf{a}_1+y\textbf{a}_2+z\textbf{a}_3$
We can create a matrix if we know how it transforms the three basis vectors

We can split up $\textbf{A}\textbf{x}=\textbf{b}$ into: $x\textbf{a}_1+y\textbf{a}_2+z\textbf{a}_3$
If the matrix $\textbf{A}$ is singular then the columns are not independent, so they all lie in the same plane. There can only be a solution where $\textbf{b}$ lies in the same plane, in which case there will be infinite solutions.

Consider $A\textbf{x}=x_1\textbf{a}_1+x_2\textbf{a}_2+x_3\textbf{a}_3+...$
The region mapped out in $R^m$ as the $x_i$ vary is called a vector space
A vector space $R^m$ is the set of $\textbf{x}$ of the form: $\textbf{x}=\lambda \textbf{u} + \mu \textbf{v} + \nu \textbf{w} + ...$
Where $\textbf{u}, \textbf{v}, ...$ are m-vectors and $\lambda, \mu, ...$ are scalars
The vectors are said to span the space

$R^m$ is itself a vector space and smaller ones within it are said to be sub-spaces of $R^m$.
The non-trivial sub-spaces of $R^3$ are a line, a plane, and the whole of $R^3$ 
The vector may be 3D but the line is a 1D object

A set of vectors which span the space and are also independent are said to form a basis and the number of these is the dimension of the sub-space

Mathematical test for independence: The vectors $\textbf{x}_i$ are linearly independent if whenever: $\lambda_1 \textbf{x}_1 + \lambda_2 \textbf{2} + ... + \lambda_n \textbf{x}_n=0$, then we must have $\lambda_1=\lambda_2=...=\lambda_n=0$

The vector space spanned by the columns of a general $m \times n$ matrix $\textbf{A}$ is called the column space of $\textbf{A}$. The dimension in the degrees of freedom sense of column space is called the rank of $\textbf{A}$.
If rank$(\textbf{A})<n$, we lose dimensions, so the matrix is singular

If we denote the columns of $\textbf{B}$ as $\textbf{b}_1, \textbf{b}_2, \textbf{b}_3,$ etc and the rows of $\textbf{C}$ as $\tilde{\textbf{C}}_1, \tilde{\textbf{C}}_2, \tilde{\textbf{C}}_3,$ etc then the multiplication of the two can be written as: $\textbf{A}=\textbf{b}_1\tilde{\textbf{c}}_1^T+\textbf{b}_2\tilde{\textbf{c}}_2^T+...$

$\textbf{x}\textbf{y}^T$ is the outer product of the two vectors, whereas the dot product is the inner product
