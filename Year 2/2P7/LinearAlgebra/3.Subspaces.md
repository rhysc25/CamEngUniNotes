Column space - the possible outputs
Left null space - the null space of $A^T$ - the outputs you can't reach
Null space - the inputs that result in 0
Row space - the inputs that give a solution

Dimension of column space = dimension of row space = rank
Column space orthogonal to left null space
Null space orthogonal to row space

The sum of dimensions of column space and the dimensions of left null space add to the number of variables in the basis vectors for them both
The sum of dimensions of row space and the dimensions of null space add to the number of variables in the basis vectors for them both

**The Least Squares Solution of Ax=b**

Say we want a quadratic fit $b = C + Dt + E t^2$
We can't find an exact solution, so we need $A\bar{\textbf{x}}$ to be as close as possible to $b$
We can do this by removing the parts of b in the left null space, by multiplying everything by $A^T$ giving $A^TAx=A^tb$. This works as $A^Tb_{lns}=0$

$A^TA$ has an inverse

**Finding an Orthonormal Set from an Arbitrary Set**
The Gram-Schmidt process

$q_1=\frac{a_1}{|a_1|}$
$\tilde{a}_2=a_2-(q_1\cdot a_2)q_1 \quad q_2 = \frac{\tilde{a}_2}{|\tilde{a}_2|}$

We can therefore introduce QR factorisation $A=QR$
Q is a matrix of orthonormal q vectors
R is a matrix of all their coefficients which make up the $a_i$ columns

R is upper triangular

Using this, we can rewrite our least squares solution $A^TA\bar{x}=A^Tb$ as $R\bar{x}=Q^T b$

To project a vector onto the column space of a matrix we just need to use the matrix $P=A(A^TA)^{-1}A^T = QQ^T$ using QR factorisation
